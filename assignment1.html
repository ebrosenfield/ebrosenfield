The circuit on my breadboard!

Here's the circuit I built. I used 220 ohm resistors for the red and yellow lights and a 100 ohm resistor for the blue light, because blue LEDs need more voltage than the other colors!

The schematic of my circuit!

Here's the schematic of my circuit. All the Blinky lights are directly connected to their own pins so that they can be controlled independently

My Arduino Code Snippet

        	
 // the setup function runs once when you press reset or power the board
void setup() {
  // initialize digital pin LED_BUILTIN as an output.
  pinMode(2, OUTPUT);
  pinMode(6, OUTPUT);
  pinMode(10, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(2, HIGH); // turn the LED on (HIGH is the voltage level)
  delay(400); // wait for 4/10th of a second
  digitalWrite(6, HIGH); // turn the LED on (HIGH is the voltage level)
  delay(400); // wait for 4/10th of a second
  digitalWrite(10, HIGH); // turn the LED on (HIGH is the voltage level)
  delay(400); // wait for 4/10th of a second
  digitalWrite(2, LOW); // turn the LED off by making the voltage LOW
  delay(400); // wait for 4/10th of a second
  digitalWrite(6, LOW); // turn the LED off by making the voltage LOW
  delay(400); // wait for 4/10th of a second
  digitalWrite(10, LOW); // turn the LED off by making the voltage LOW
  delay(400); 
}
